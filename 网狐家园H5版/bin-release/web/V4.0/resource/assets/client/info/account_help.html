<记住密码>

勾选“记住密码”选项，将记住最近一次登录所输入的密码，在下次登录时，将无需再次输入密码。

<自动登录>

勾选“自动登录”选项，“记住密码”选项也会自动被勾选，系统将记住帐号密码并在下次进入游戏时自动完成登录过程。

<登录异常>

导致登录异常的原因很多，请根据以下方式进行排查：

1. 手机内存不足或同时运行的手机应用过多，可能会导致游戏无法正常登录。建议您在运行游戏程序时，尽量将不必要的应用关闭，保证手机有足够的运行空间。

2. 网络环境不佳导致无法正常登录。建议选择Wi-Fi方式或是在网络环境良好的条件下进行游戏。在确保网络正常的情况下还是无法正常登录，可以尝试重新启动游戏。

3. 以上方法均不能解决您的登录问题，请联系客服寻求帮助。给您造成的不便，敬请谅解！

<异步登录>

同一游戏帐号不能同时在多个终端上进行游戏。如：某游戏帐号正在一终端进行游戏，其它终端用相同帐号将不允许再进行游戏。
